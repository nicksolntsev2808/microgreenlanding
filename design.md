# Design System — Грін гарден

## Inspiration
Editorial organic food brand — cream backgrounds, serif display fonts, generous white space, photography-first layout, minimal UI chrome.

## Color Palette
- `--cream`: #F5F2EB (page background)
- `--ink`: #1A1A14 (primary text)
- `--moss`: #3B5040 (accent green — CTA buttons, highlights)
- `--sage`: #8AA68B (muted green for secondary elements)
- `--warm-gray`: #C8C4BB (borders, dividers)
- `--highlight`: #E8F0E4 (light green tint blocks)

## Typography
- Display / headings: **Playfair Display** (serif, dramatic, editorial)
- Body / UI: **Jost** (clean geometric sans-serif)
- Sizes: hero 72-80px, section titles 48-56px, card titles 24px, body 16-18px
- Line height: generous — 1.5-1.7 for body, 1.1 for display

## Layout
- Max content width: 1280px
- Generous section padding: 100-120px vertical
- Asymmetric hero — text left, product image right
- Full-bleed image sections with text overlay
- Table with alternating cream/white rows

## Motion
- Page load: staggered fade-in-up on hero elements
- Section reveals: intersection observer fade-in
- Hover on CTA: gentle scale + bg shift

## Components
- CTA button: moss green fill, cream text, no border radius (square-ish, 4px max)
- Variants radio: subtle bordered options
- Nav: minimal, top — logo center, links left/right, transparent on scroll
- Footer: dark ink background, cream text
